A simple story style image feature built with react. 

This project lets users add short-lived image stories, view them full-screen, and move through them naturally with timed progress.
 
## Features
- Upload images as stories
- Stories automatically expire after 24 hrs
- Stories auto-advance every 3 secs
- Tap left/right areas to navigate between stories
- Data stored locally using browser storage

## Built With
- React (Vite)
- React Router
- Tailwind CSS
- JavaScript
- LocalStorage API
